# toon_video_transformer

ToonVideo Transformer는 비디오를 카툰 스타일로 변환하는 간단하고 효과적인 도구입니다. 이 프로그램은 사용자가 선택한 비디오 파일을 처리하여 각 프레임에 카툰 스타일 효과를 적용합니다. "ToonVideo Transformer"를 사용하면 사용자는 자신의 비디오를 간단하게 카툰 스타일로 변환하여 독특하고 재미있는 비주얼 효과를 얻을 수 있습니다.

## 한계
사물이나 인물의 윤곽이 뚜렷하고 명확한 경우, 적응 임계값을 사용한 엣지 감지가 잘 작동하여 카툰 효과를 강조시킬 수 있고 양방향 필터를 사용하여 색상을 보존하면서도 노이즈를 감소시키기 때문에 색채가 다양하고 풍부한 영상에서는 카툰 효과가 더 잘 드러날 수 있습니다. 반면에 윤곽이 모호하거나 흐릿한 경우 엣지 감지가 제대로 작동되지 않아 카툰 효과가 부각되지 않을 수 있고 색상의 변화를 강조하는 경향이 있기 때문에 단조로운 영상에서는 효과가 미미할 수 있습니다.
