# toon_video_transformer

ToonVideo Transformer는 비디오를 카툰 스타일로 변환하는 간단하고 효과적인 도구입니다. 이 프로그램은 사용자가 선택한 비디오 파일을 처리하여 각 프레임에 카툰 스타일 효과를 적용합니다. 강한 윤곽과 대비, 생생한 색상 강조, 그리고 간단한 패턴과 질감을 특징으로 하는 만화 스타일의 영상을 생성합니다. "툰비디오 변환기"를 사용하면 사용자는 자신의 비디오를 간단하게 카툰 스타일로 변환하여 독특하고 재미있는 비주얼 효과를 얻을 수 있습니다.
